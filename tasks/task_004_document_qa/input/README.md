# Document QA Fixture

This task simulates answering operational questions from a small internal document set.

Useful files:
- `brief.txt` contains the visible request
- `docs/operations.md` describes incident process ownership and response targets
- `docs/incident_roles.md` describes who handles communications during major incidents
- `docs/quarterly_plan.md` contains KPI goals
- `docs/team_directory.md` adds organization context that may support synthesis
