# Data Analysis Fixture

This task simulates a lightweight business analysis workflow.

Useful files:
- `brief.txt` explains the stakeholder question
- `data/sales.csv` contains the raw dataset
- `data_dictionary.md` defines the columns and the ranking rule

Recommended approach:
- inspect the dataset
- compute the month-over-month revenue delta per category
- rank categories by absolute growth
- produce a short stakeholder-friendly summary
