# Research Protocol

For this task:

1. Start with broad search to identify the current milestone and the likely primary sources.
2. Cross-check claims across at least 2 independent sources.
3. Prefer official EU or regulator sources for milestones and obligations.
4. Use secondary reporting only to add context, not as the only evidence.
5. Include source dates where available, because timing matters.

The benchmark is evaluating research process quality as much as factual coverage.
